# ⚡ Circuit Rush

Circuit Rush es un juego web desarrollado con **React**, **Vite** y **Tailwind CSS**, donde el jugador debe poner a prueba sus reflejos, memoria y velocidad para completar diferentes niveles antes de que el tiempo se agote.

## 🎮 ¿Cómo se juega?

- Sigue el color objetivo indicado en la pantalla.
- Haz clic únicamente en los nodos correctos.
- Evita las sobrecargas y los nodos incorrectos.
- Aprovecha los potenciadores para obtener más puntos.
- Supera los 5 niveles y consigue la mayor puntuación posible.

## ✨ Características

- 🎯 5 niveles con dificultad progresiva.
- ⚡ Sistema de puntuación y combos.
- ❤️ Sistema de vidas.
- ⏱️ Temporizador por nivel.
- 🌙 Modo oscuro.
- 🔊 Efectos de sonido.
- 🏆 Almacenamiento del récord utilizando LocalStorage.
- 🎉 Pantalla de resultados con estadísticas finales.

## 🛠️ Tecnologías utilizadas

- React
- Vite
- Tailwind CSS
- JavaScript (ES6+)
- Lucide React

## 🚀 Instalación

```bash
git clone https://github.com/Fioo1/circuit-rush-v.git
cd circuit-rush-v
npm install
npm run dev
```
